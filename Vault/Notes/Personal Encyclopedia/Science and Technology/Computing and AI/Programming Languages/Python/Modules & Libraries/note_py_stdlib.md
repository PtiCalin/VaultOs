### 📚 The Python Standard Library

The **Standard Library** is a collection of modules that ship with Python—no extra installs needed.

---

### 🧰 Some Essentials

- `os`, `sys`, `pathlib` – file/system
- `math`, `random`, `statistics`
- `datetime`, `time`
- `json`, `csv`, `pickle`
- `re`, `functools`, `collections`
