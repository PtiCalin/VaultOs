## 🔤 Python Strings 101

Strings are sequences of characters, defined with single or double quotes.

```python
msg = "Hello"
print(len(msg))  # 5
```

---

### 🧰 Common Methods

- `.lower()`, `.upper()`
- `.strip()`, `.replace()`
- `.split()`, `.join()`
- `.startswith()`, `.endswith()`

- f-strings

- Regular expressions
