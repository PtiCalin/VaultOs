## â• Python Operators

Operators perform operations on variables and values. Types include:

- Arithmetic: `+`, `-`, `*`, `/`, `**`, `%`
- Comparison: `==`, `!=`, `>`, `<`, `>=`, `<=`
- Logical: `and`, `or`, `not`
- Assignment: `=`, `+=`, `*=`, etc.
- Bitwise: `&`, `|`, `^`, `~`, `<<`, `>>`
- Membership: `in`, `not in`
- Identity: `is`, `is not`
